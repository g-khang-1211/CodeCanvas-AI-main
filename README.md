# CodeCanvas AI

An AI-powered study platform that helps you learn faster by generating lessons, flashcards, quizzes, and study materials on demand.

You choose what you want to learn. The app builds the learning content for you.

---

## What This App Does

This app turns topics into structured learning experiences using AI.

You can:

* Study new subjects
* Review concepts with flashcards
* Learn at your own pace
* Use your own AI API key for full control

---

## Core Features

### AI-Generated Learning Content

* Full lessons for any topic
* Clear explanations
* Structured sections
* Examples and summaries

### Flashcards

* Automatically generated flashcards
* Designed for quick review
* Adapted to difficulty level

### Quizzes

* Practice questions per lesson
* Multiple question formats
* Immediate feedback

### Courses and Units

* Subjects are organized into courses
* Courses contain levels
* Levels contain units
* Units contain lessons and quizzes

---

## Personal API Key Support

You bring your own AI API key.

* Your key is private
* You control usage
* You can change or remove it anytime
* If saving fails, the key still works for the current session

---

## Accounts and Sign-In

* Sign in with Google
* No passwords to remember
* Your progress stays linked to your account
